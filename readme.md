# Shadow Carbon API 🌿

A lightweight carbon footprint estimator API built with FastAPI, aligned with the **CSRD (Corporate Sustainability Reporting Directive)** framework.

Built as an MVP course project to demonstrate ESG-ready carbon emission estimation from everyday transaction categories.

---

## What it does

Takes a spending amount and category (fuel, groceries, transport, utilities) and returns an estimated carbon footprint in kg CO₂e, along with GHG scope classification and CSRD reporting category.

## Setup & Running

**1. Install dependencies**
```bash
pip install fastapi uvicorn pydantic
```

**2. Run the server**
```bash
python -m uvicorn main:app --reload
```

**3. Open the UI**
```
http://127.0.0.1:8000/ui/
```

**4. Or explore the API docs (auto-generated by FastAPI)**
```
http://127.0.0.1:8000/docs
```

## API Endpoint

### `POST /estimate-carbon`

**Request body:**
```json
{
  "amount": 5000,
  "category": "fuel",
  "country": "PK"
}
```

**Response:**
```json
{
  "estimated_carbon_kg": 45.0,
  "category": "fuel",
  "scope": "Scope 3",
  "framework": "CSRD",
  "reporting_category": "CSRD - Financed / Indirect Emissions",
  "confidence": "low (proxy-based estimate)",
  "esg_ready": true
}
```

## Supported Categories

| Category   | Emission Factor | GHG Scope | CSRD Category                        |
|------------|----------------|-----------|--------------------------------------|
| Fuel       | 0.009 kg CO₂e  | Scope 3   | CSRD - Financed / Indirect Emissions |
| Groceries  | 0.002 kg CO₂e  | Scope 3   | CSRD - Financed / Indirect Emissions |
| Transport  | 0.007 kg CO₂e  | Scope 3   | CSRD - Financed / Indirect Emissions |
| Utilities  | 0.005 kg CO₂e  | Scope 2   | CSRD - Energy Emissions              |
